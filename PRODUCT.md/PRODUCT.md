# AI Coding Hub - 产品需求文档

## 产品定位

**一个账号，用遍所有 AI 编程模型**

目标用户：程序员、开发者、技术团队

核心价值：
- 省钱：自动选择最优价格模型
- 省事：一个账号管理所有模型
- 高效：统一使用量和账单统计

## 技术架构

- **前端**: Next.js + Tailwind CSS
- **后端**: Python FastAPI
- **数据库**: PostgreSQL
- **部署**: Vercel + Railway
